导入数据库文件标准说明：
1.每组实验文件名即为数据库中试验编号；
2.相对保留时间导入文件中，每个txt文件中仅包含1组试验数据；
3.峰面积导入文件中，每个txt文件中包含4组试验数据，并以实验波长递增的顺序排列4组试验
4.光谱吸收导入文件中，每个.xlsx(.xls)文件从第二列起，每一列为一组试验，每组试验对应的第一行中数据为试验编号。若有多个波峰，以‘/’分隔。
5.文件中除# ‘ ’ , 外，无特殊字符。
6.企业版级联检测对照品库 JSON 额外支持以下字段：
   - rt / rrt：主峰保留时间 / 相对保留时间（rrt 优先，缺失时可用 rt 与参照物 rt 计算）
   - lambda1 / lambda2：紫外最大吸收波长，缺失或空值用 "/" 或 null 表示
   - a245 / a250 / a255 / a260：四个固定波长下的峰面积常数，用于 Step 3 峰面积比计算
（具体导入样本文件见附件）
